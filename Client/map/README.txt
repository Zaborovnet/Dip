Разбор загрузки картинок через QGeoView

Загрузка картинок 
Загрузка картинок происходит в файле tringle.cpp с использованием оператора switch для загрузки несколькихз картинок. Непосредственно для загрузки используется
функция либы QGeoView/QGVImage под названием LoadImage(url).

Вывод изображения
Вывод изображения прпоисходит в файле flags.cpp через содание списка и в нём трёх новых элеметов Tringle со следующими аргуметами:
1. Координаты где будет располагаться картинка (координаты обычные long late)
2. Выбор из свича нужной картинки 
После этого создаётся createQGVImage().toImage() с помощью котрогоо мы будем выводить саму картинку. Дальше не сложными махинациями отделяем элеметы картинок от
других (это сделано для нормального вывода излбражения, так как вывод rectangle и tringle немного отличается), после всего этого, выкидываем элементы на карту.
