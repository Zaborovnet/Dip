Сервер для генерации координати присваивания их к каждой цели и выбора случайной фотографии из папки.
Данный сервер должен отпралять данные в таблицу базы данных postgress. 
На вход картинка.
На выход сгенерированный наомер цели, её случайные координаты и передача номера цели серверу для нейросети (NNServer) (чтобы он понимал
с какой целью он работает и вправильную строку передал тип ВО (воздушного объекта)). Так же переда случайно взятой каринки на NNServer.
